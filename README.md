# example-react-image-viewer

This is an example of image viewer with zoom in/out, drag on React.

## Development

```shell
# Install dependencies
$ npm install

# Run development server
$ npm run dev
```

## Deploy

```shell
# Build assets
$ npm run build
$ git commit -A
$ git push origin main
```
